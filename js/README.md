# PDF to Excel Smart Extractor

A browser-based tool for extracting structured data from PDFs and exporting to Excel.

## Features
- Drag & drop PDF upload (single or multiple)
- Smart field extraction with fuzzy matching
- Dynamic header configuration
- Excel-like data preview
- Export to formatted Excel files
- Template save/load functionality

## How to Run Locally

1. Clone or download this repository
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge)
3. No server required - runs entirely in the browser!

## Usage

1. **Upload PDFs**: Drag and drop PDF files or click to browse
2. **Configure Headers**: Add/remove/rename the fields you want to extract
3. **Extract Data**: Click "Extract Data" to process all uploaded PDFs
4. **Review & Edit**: Check the extracted data in the preview table
5. **Export**: Click "Export Excel" to download the results
